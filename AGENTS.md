# Agent Developer Guide (AGENTS.md)

Welcome, AI Agent! This repository is designed to be agent-friendly. 

Please read the following documents before writing code or adding features:

1. **[DEVELOPING_AGENTS.md](DEVELOPING_AGENTS.md)**: The main instruction guide detailing how to build new SVG generators, register them in the dynamic navigation, and adhere to vector coordinate rules.
2. **[generators.json](generators.json)**: The central registry of all active tools.
3. **[shared-nav.js](shared-nav.js)**: The navigation utility imported by all generators.

Thank you for building with us!
