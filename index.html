Tips for Import & Cutting

1. Keep Page Dimensions Match

When generating an SVG, ensure the physical height and width match your intended workpiece size. Write down these dimensions to scale the object correctly inside Cricut Design Space.

2. Red vs Black Vectors

Some tools allow rendering helper paths or guides. When importing, you can map specific stroke colors to 'Draw/Pen' operations and others to 'Cut' operations.

3. Weld/Attach in Design Space

After importing the SVG, remember to select all elements and click "Attach" so that the relative positions of the cut lines are preserved during the cutting process.
